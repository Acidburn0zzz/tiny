== Frequently Asked Questions ==

= How do I enable and customize breadcrumbs? =

To enable breadcrumbs, the theme requires you to either use the `tiny_breadcrumbs` filter or [Yoast SEO](https://wordpress.org/plugins/wordpress-seo/). Just install and activate the plugin.
